# Fast Nitro Rally Minecraft Java Edition

Fast Nitro Rally представляет собой аркадную гоночную игру, реализованную на языке Python для среды Minecraft. Проект поставляется в формате готового решения со встроенным локальным сервером, что исключает необходимость самостоятельного поиска и установки модификаций.

## Структура проекта

* server — Директория с ядром локального сервера Minecraft. Содержит предустановленные конфигурации и программные интерфейсы для связи с Python.
* Start_Server.lnk — Файл для быстрого запуска серверной части.
* mcpi — Локальная библиотека API для передачи команд от скрипта к игровому миру.
* race.py — Основной программный файл, содержащий игровую логику, систему обработки коллизий и генерацию уровней.
* input_system.py — Модуль для обработки нажатий клавиш и управления транспортным средством.
* requirements.txt — Список внешних зависимостей Python.
* LICENSE — Текст лицензионного соглашения.

## Технические требования

* Minecraft Java Edition версии 1.16.5 или выше.
* Python версии 3.7 или выше.
* Операционная система Windows для работы стартового файла сервера.

## Инструкция по запуску

1. Запуск сервера: Откройте файл Start_Server.lnk и дождитесь окончания процесса загрузки мира в консоли.
2. Подключение к игре: Запустите клиент Minecraft и подключитесь к серверу по адресу localhost.
3. Запуск игрового скрипта:
   * Установите зависимости: pip install -r requirements.txt
   * Запустите скрипт: python race.py
4. Начало игры: После запуска скрипта в мире Minecraft вокруг персонажа будет автоматически выстроена гоночная трасса.

## Характеристики транспортных средств

В игре реализовано три типа объектов с уникальными параметрами:

* Мотоцикл: Размер 1х1х1 блок, высокая скорость, 20 очков за обход.
* Легковой автомобиль: Размер 1х1х2 блока, средняя скорость, 10 очков за обход.
* Грузовик: Размер 2х2х3 блока, низкая скорость, занимает две полосы движения, 5 очков за обход.

## Игровые механики и системы

### Система бонусов
На трассе случайным образом появляются золотые блоки (монеты). Сбор такого блока приносит игроку 50 дополнительных очков. При активации системы ускорения ценность собранных монет удваивается.

### Искусственный интеллект трафика
Транспортные средства под управлением компьютера оснащены системой предотвращения столкновений. Каждая машина сканирует пространство перед собой и при обнаружении препятствия снижает скорость или останавливается до момента освобождения пути.

### Система Nitro
Активируется клавишей Стрелка Вверх. При активации скорость игры увеличивается в два раза на 3 секунды. Визуально это отображается изменением материала машины на красную шерсть и появлением следа из блоков красного камня. После использования система требует 8 секунд на перезарядку.

### Динамическое окружение
Для создания эффекта высокой скорости реализован алгоритм цикличного обновления декораций. По краям трассы в реальном времени генерируются элементы растительности, заборы и источники света (светящийся камень), обеспечивающие видимость в ночное время.

### Прогрессия и жизни
У игрока имеется запас из 3 жизней. При столкновении с препятствием одна жизнь теряется, а объект столкновения удаляется из мира. Сложность игры постепенно увеличивается за счет сокращения интервалов между появлением новых машин.

## Управление

* Стрелки Влево / Вправо — Смена полосы движения.
* Стрелка Вверх — Активация Nitro.
* Клавиша Esc — Остановка выполнения скрипта.

## Параметры конфигурации

Основные настройки баланса могут быть изменены в начале файла race.py:
* ROAD_WIDTH — Ширина дорожного полотна.
* BASE_GAME_SPEED — Базовая задержка игрового цикла (скорость).
* SPAWN_POOL — Вероятность появления различных типов машин.
