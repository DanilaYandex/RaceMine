# Fast Nitro Rally (Minecraft Java Edition)

**Fast Nitro Rally** — это динамичная аркадная гонка, реализованная на языке Python и интегрированная прямо в мир Minecraft. Проект представляет собой экосистему «всё в одном»: вам не нужно искать сторонние моды, так как в репозиторий уже включен настроенный сервер с необходимым программным интерфейсом.

---

## Структура репозитория

* **`server/`** — папка с предустановленным ядром Minecraft-сервера. Содержит все необходимые плагины и конфигурации для взаимодействия с Python-скриптами.
* **`Start_Server.lnk`** — ярлык для быстрого запуска локального сервера в один клик.
* **`mcpi/`** — локальная библиотека API, которая служит «мостом» между вашим кодом и миром игры.
* **`race.py`** — основной исполняемый файл игры. Отвечает за логику трассы, спавн машин и правила гонки.
* **`input_system.py`** — вспомогательный модуль для захвата нажатий клавиш клавиатуры.
* **`requirements.txt`** — список необходимых Python-зависимостей.
* **`LICENSE`** — юридическая информация об использовании кода.

---

## Технические требования

* **Minecraft Java Edition**: Версия 1.16.5 или выше.
* **Python**: Версия 3.7 или выше.
* **ОС**: Windows (для корректной работы ярлыка запуска сервера).

---

## Инструкция по запуску

1.  **Запуск сервера**: Откройте файл `Start_Server.lnk`. Дождитесь полной загрузки мира в открывшейся консоли.
2.  **Подключение**: Запустите свой Minecraft и зайдите в сетевую игру по адресу `localhost`.
3.  **Запуск игры**: В терминале (командной строке) выполните следующие команды:
    ```bash
    pip install -r requirements.txt
    python race.py
    ```
    После этого в мире Minecraft вокруг вашего персонажа автоматически построится гоночная трасса.

В игре реализовано три класса объектов, которые отличаются габаритами, скоростью и ценностью при обгоне:

| Тип авто | Габариты (Ш х В х Д) | Очки | Особенности |
| :--- | :--- | :--- | :--- |

| **Легковое авто** | 1 х 1 х 2 | 10 | Стандартный противник, встречается чаще всего. |
| **Грузовик** | 2 х 2 х 3 | 5 | Медленный, но занимает 2 полосы движения. |

## Игровые системы и интеллект

### 1. Интеллект трафика (AI Intelligence)
Машины в игре обладают логикой предотвращения столкновений:
* **Сканирование пути**: Каждый автомобиль проверяет пространство перед собой.
* **Предотвращение заторов**: Если перед машиной находится другой объект, она замедляется или ждет, пока путь освободится, чтобы не создавать завалы из блоков.

### 2. Система Nitro
Активируется клавишей **Стрелка Вверх**:
* **Механика**: Скорость игры увеличивается вдвое на 3 секунды.
* **Визуализация**: Машина игрока меняет цвет (используется красная шерсть), а сзади появляется след из блоков красного камня.
* **Бонус**: Все очки, полученные во время действия нитро, удваиваются.
* **Перезарядка**: После использования система Nitro недоступна в течение 8 секунд.

### 3. Динамическое окружение (Scenery)
Для создания иллюзии движения используется алгоритм обновления декораций:
* **Дорожная разметка**: Белые линии на асфальте циклично перемещаются.
* **Генерация объектов**: По краям дороги автоматически создаются деревья, заборы и работающие фонари из светящегося камня.

### 4. Система жизни и бонусов
* **Здоровье**: У игрока есть 3 жизни. При столкновении вы теряете одну жизнь, а вражеская машина уничтожается.
* **Монеты**: На дороге могут появляться золотые блоки (монеты), сбор которых дает +50 очков.

## Управление

* **Стрелки Влево / Вправо** — маневрирование между полосами.
* **Стрелка Вверх** — активация Nitro.
* **Esc** — экстренная остановка скрипта.
